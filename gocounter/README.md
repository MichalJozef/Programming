# Overview

Go implementation of finding duplicate strings in text input.

# Usage

./godif <text1>, <text2> ... <textn>
  
# Tests

Processor 3,4 GHz Quad-Core Intel Core i5 8 GB 2400 MHz DDR4

Files: mock2.txt (3.8G) mock3.txt (520M)

Elapsed time:  8.241735571s

Case: test files contain duplicate lines as much as possible on testing purpose to prove "worst case" testing scenario time/space complexity
